# Контрольная работа №1

## ToDo App


## **задание:**
- ` Необходимо сделать ToDo приложение которая отслеживает выполнение таска`

## **функционал:**
- Default CRUD (create, read, update, delete)
- страница со всеми тасками
- страница которая выводит только выполненные
- страница котора явыводит невыполненные
- возможность помечать таски как помеченные
- помеченные таски выводить с датой выполнение

## Доп функционал:

- вывод времени выполнения таска (от момента создания до момента выполнения)
- что то типо прогрессбара можно просто в виде процентов выполненных и не выполненных тасков
